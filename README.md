# Learning Golang

A structured self-study repo documenting my journey learning Go from scratch.

I'm learning Golang through hands-on practice:
- one concept at a time,
- one commit at a time.

Each folder covers a core concept with working code examples and notes.

## What's covered so far
- 01 Basics — variables, types, arrays, slices, input
- 02 Loops & conditionals — for, switch, if/else patterns
- 03 Functions — parameters, multiple returns, named returns
- 04 Structs — basic, nested, struct with functions
- 05 Methods — basic methods, pointer receivers
- 06 Packages — fmt, os, bufio, time, third-party
- 07 Error handling — basic errors, error returns

## Mini projects built alongside
- `calculator` — basic calculator using functions
- `student` — student struct with methods

## Goals
- Get comfortable enough with Go to build real CLI tools and small backend services
- Apply Go in portfolio projects alongside Python-based ML/AI work

## Stack
Go 1.21+ · gopsutil v3 · No frameworks · Pure standard library focus
