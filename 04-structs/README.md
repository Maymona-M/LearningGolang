# 04-structs - Go Structs & Data Modelling

This folder introduces structs in Go, allowing related data to be grouped into custom data types. Structs are one of Go's most fundamental features and form the basis of data modelling in real-world applications.

## Topics Covered

- Creating structs
- Initializing struct instances
- Accessing struct fields
- Passing structs to functions
- Nested structs (structs within structs)

## Purpose

This section introduces data modelling in Go using structs. It teaches how to organize related data into custom types, pass structs between functions, and represent more complex relationships using nested structs. These concepts provide the foundation for methods, interfaces, and larger applications such as APIs and system monitoring tools.
 and more realistic data models.
