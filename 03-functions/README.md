# 03-functions - Go Functions Practice

This folder covers the fundamentals of functions in Go.

## Topics Covered

- Basic functions
- Functions with parameters
- Functions with return values
- Multiple return values
- Function composition (mini calculator project)

## Purpose

This section introduces functional decomposition in Go. It teaches how to organize code into reusable functions, handle input and output through parameters and return values, and build small modular programs. These concepts form the foundation for backend development and system tools such as the system monitor project.

## Notes

- All files use `package main` for simplicity.
- Each file is independent and can be run using:

```bash
go run filename.go
```
