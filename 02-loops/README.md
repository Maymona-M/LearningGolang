# 02-loops - Go Control Flow

This folder covers control flow in Go.

## Topics Covered

- if / else statements
- switch statements
- for loops (Go's only loop)
- while-style loops using `for condition`
- nested loops

## Key Idea

Go does NOT have:
- no `while`
- no `do-while`

Everything is done using `for`.

## Examples

Run any file individually:

```bash
go run filename.go
```
